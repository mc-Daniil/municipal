## Решение

В приложенной программе реализован шифр Цезаря. Достаточно лишь перебрать небольшое количество возможных ключей и выбрать подходящий, используя условие: "В правильно расшифрованном флаге содержится слово `correct`"

Так как любой сдвиг букв в обратную сторону аналогичен некому обычному сдвигу, при переборе для расшифровки можно переиспользовать функцию из задания. Скрипт расшифровки приложен в [solve.py](solve/solve.py).
